# Fin-9 Web

Pagina web simples em Vite para atender a exigencia da Play Store sobre exclusao de conta e dados.

## Executar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Configuracao opcional

Copie `.env.example` para `.env` se quiser exibir um e-mail de suporte na pagina:

```bash
VITE_SUPPORT_EMAIL=suporte@seu-dominio.com
```
